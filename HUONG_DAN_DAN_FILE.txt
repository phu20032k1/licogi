LICOGI 18.3 MVP - CÁCH DÁN FILE

TRƯỜNG HỢP 1: BẠN ĐÃ CÓ PROJECT NEXT.JS
1. Giải nén file licogi_mvp_files.zip.
2. Copy thư mục "src" trong gói này.
3. Dán đè/merge vào thư mục "src" của project Next.js.
4. Mở Terminal đúng tại thư mục project và chạy:

   npm install leaflet react-leaflet
   npm install -D @types/leaflet

5. Sau đó bạn tự chạy:

   npm run dev

6. Mở trình duyệt:

   http://localhost:3000


TRƯỜNG HỢP 2: CHƯA CÓ PROJECT NEXT.JS
Tạo project bằng lệnh:

npx create-next-app@latest licogi-os --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"

Sau đó:
- Vào thư mục licogi-os.
- Copy thư mục src trong gói này và dán đè/merge vào project.
- Chạy hai lệnh cài thư viện ở trên.
